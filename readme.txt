1. Создать примитивный блог.
 1.0 Пользователь должен иметь возможность создавать статьи(если залогинен).
 1.1 Реализовать авторизацию пользователя(дополнительно - продумать регистрацию и система ролей).
 1.2. На главной странице - выводить все статьи отсортированными по дате( дополнительно - сортировка по всем полям).
 1.3. Удаление статьи. Пользователь должен уметь удалять свои статьи(дополнительно пользователь должен иметь возможность модифицировать свои статьи, админ может делать всё).
